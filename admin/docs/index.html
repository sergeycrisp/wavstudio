How to use the template

  1. To use this template you'll need some understanding of HTML and CSS. If you are developing your own admin panel then this can help you improve the way it looks and provide you with different easy to use components to use.

  2. The easiest way to start using the theme is to start coding your project, whether its ReactJS, AngularJS or something else and copying the layout in index.html to it. Then you should copy the HTML from home.html file so you have the menu sections you need.

  3. For each page you need to create, you can copy the elements found in the example theme.


Changing colours

The theme uses the default UIKit styles to control the colors. You should use theme.less to set up the default theme styles. Other external styles not related directlyy to UIKit are found in styles.less. You'll need to compile styles.less and theme.less in order to customize the theme. If you want to keep the same styles and just make changes to the sidebar and wrappers you must use only styles.less.


Customizing the HTML

This theme uses two main sections, a sidebar #sidebar-wrapper and the page content #page-content-wrapper. The '.uk-container' is where your styles for different pages will go. Modals are located just before the 'footer'. The '.uk-nav' is used to show the menu of the sidebar, where 'uk-open' opens a menu item.
